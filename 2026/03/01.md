## 分かったこと
```
const Animal = [
{name:"レオ",age:4},
{name:"ハナ",age:7},
{name:"ララ",age:4}]
console.log(Animal[2].name);
>ララ
```
この文型は配列にオブジェクトを入れているが、メリットとして、オブジェクトをインデックス番号で呼び出せるということがある。
```
const Animal = [
{name:"レオ",age:4,},
{name:"ハナ",age:7},
{name:"ララ",age:4},
{name:()=>{
console.log("ララ")},
age:4}
]
Animal[3].name()
>ララ
```
前述の方法と変わらないが、オブジェクトのプロパティには関数が入れられるので、オブジェクトに関数を入れて、処理をさせ、それをインデックスで呼び出せるという手法が取れる。
