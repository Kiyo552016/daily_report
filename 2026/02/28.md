## 分かったこと
```
js
const Animal = {
name:"John",
age:4,
greet:function(){
console.log(`こんにちは${this.name}さん`);
}
}
Animal.greet();
>こんにちはJohnさん
```
thisの挙動は数種類あるようであるが、私が理解したパターンではこのthisはAnimalを指している。「何がthisなのか」=この定数自身という解釈が良いように思う。例外もあり、まだ飲み込めていない。
