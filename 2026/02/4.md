## わかったこと
３つのテーブル（表）の結合方法
```
SELECT　カラム名(列)
FROM　テーブル名
JOIN　繋げたいテーブル名
ON　　テーブル名.結合キー(カラム名) = 繋げたいテーブル名.結合キー(カラム名)
JOIN　繋げたいテーブル名その２
ON　テーブル名.結合キー = 繋げたいテーブル名その２.結合キー
``` Progate SQL JOIN,サブクエリ編修了、進捗率76%
